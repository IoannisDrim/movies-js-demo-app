# Movie Rama

**Movie Rama** is a demo application designed to showcase my skills in modern web development. This project is still under implementation, but it provides a first impression of the technologies and techniques I use.

## Purpose

The primary purpose of this application is to demonstrate my proficiency in:

- JavaScript
- Webpack
- Babel
- ESLint
- Jest
- CSS/Sass
- HTML
- Vercel deployment

This project is intentionally built using only JavaScript to demonstrate strong proficiency with the language. The goal is to show that understanding JavaScript is fundamental, as all modern frameworks are built on top of it.

## Technologies Used

- ![JavaScript](https://img.icons8.com/color/48/000000/javascript.png) **JavaScript**: The core programming language used for the application.
- ![Webpack](https://img.icons8.com/color/48/000000/webpack.png) **Webpack**: Module bundler to manage and bundle the assets and modules.
- ![Babel](https://img.icons8.com/color/48/000000/babel.png) **Babel**: JavaScript compiler to use the latest JavaScript features.
- ![ESLint](https://img.icons8.com/color/48/000000/eslint.png) **ESLint**: Linting utility to ensure code quality and consistency.
- ![Jest](https://img.icons8.com/color/48/000000/jest.png) **Jest**: Testing framework to write and run tests.
- ![Sass](https://img.icons8.com/color/48/000000/sass.png) **CSS/Sass**: Styling the application with CSS and Sass preprocessor.
- ![HTML](https://img.icons8.com/color/48/000000/html-5.png) **HTML**: Markup language for structuring the web pages.
- ![Vercel](https://img.icons8.com/color/48/000000/vercel.png) **Vercel**: Deployment platform to host the application.

## Deployment

The application is deployed on Vercel, providing a seamless and efficient way to showcase the demo. You can access the live version of the app [here](https://movies-js-demo-app.vercel.app/).

## Getting Started

To get started with the project locally, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://github.com/IoannisDrim/movies-js-demo-app.git
   ```

2. Navigate to the project directory:
   ```sh
   cd movies-js-demo-app
   ```

3. Install the dependencies:
   ```sh
   npm install
   ```

4. Run the development server:
   ```sh
   npm start
   ```

5. Build the project for production:
   ```sh
   npm run build
   ```
