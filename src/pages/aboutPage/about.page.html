About MovieRama

A demo application built entirely with vanilla JavaScript — no frameworks, no libraries, no magic. Just ES6+ classes, the native DOM API, and a deliberately structured component architecture to prove that reactivity, routing, and clean UIs do not require a framework to implement well.

Architecture patterns

  • Hash-based client-side router with page lifecycle (render / afterRender / unmount)
  • Class-based component model with explicit mount and teardown
  • Frozen singleton pattern for shared services (Modal, Loader)
  • Infinite scroll via throttled CustomEvent dispatch
  • Safe DOM templating — no eval, no innerHTML injection
  • Accessible modal with ARIA attributes, focus trap, and focus restoration

Data

Movie data is provided by the TMDB API (The Movie Database).
